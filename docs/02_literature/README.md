# Literature (문헌 연구)

목표는 “많이 읽기”가 아니라 **제안서/연구기획에 바로 들어가는 근거**를 축적하는 것입니다.

## 폴더 구성

- `briefings/`: (추천) 기술 동향/브리핑/슬라이드 요약 (제안서 근거로 바로 쓰는 정리본)
- `papers/`: (선택) PDF 보관
- `reading-notes/`: 논문 1편당 1개 메모(템플릿 기반)
- `synthesis/`: 주제별 종합 노트(여러 논문을 묶어서 결론/갭/우리 전략 도출)
- `library.bib`: BibTeX 참고문헌(선택)

## 워크플로우(권장)

1. 논문 확보 → `papers/`에 저장(원하면)
2. 논문 메모 생성 → `reading-notes/`에 템플릿 복사
3. 월/주 단위로 주제별 종합 → `synthesis/`에 업데이트
4. 제안서 주장(차별성/필요성/기술우위)에 매핑

## 파일 네이밍(권장)

- 논문 메모: `YEAR-FirstAuthor-ShortTitle.md`  
  예) `2024-Smith-EventBasedLearning.md`

## Templates

- 논문 메모 템플릿: `reading-notes/_template.md` (또는 `templates/paper_note.md`)
- 주제별 종합 템플릿: `synthesis/_template.md`
