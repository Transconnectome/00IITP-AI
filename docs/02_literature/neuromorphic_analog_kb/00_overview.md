# Neuromorphic & Analog Computing for Brain Foundation Models: Expanded Overview

## 1. Executive Summary

This knowledge base has been expanded to 20+ key references (2024-2025) bridging **Brain Foundation Models (BFMs)** / **Digital Twin Brains (DTB)** and **Neuromorphic & Analog Computing**. It provides a comprehensive library for NotebookLM to support the SNU Connectome Lab's grant proposal.

## 2. Expanded Reference Library (Categorized)

### 2.1 Surveys & Strategic Roadmaps

* **[Brain Foundation Models: A Survey (Zhou 2025)](./pdfs/zhou_2025_brain_foundation_models.pdf)**: The definitive 2025 survey on BFMs.
* **[Neuromorphic Roadmap (Mehonic 2024)](./pdfs/mehonic_2024_neuromorphic_roadmap.pdf)**: A 90-page guide to emerging neuromorphic technologies.

### 2.2 Brain Foundation Models (BFM)

* **[CSBrain (2025)](./pdfs/csbrain_2025_eeg_fm.pdf)**: Cross-scale spatiotemporal model for EEG.
* **[BrainOmni (2025)](./pdfs/brainomni_2025_eeg_meg_fm.pdf)**: Unified BFM for EEG and MEG signals.
* **[CBraMod (2024)](./pdfs/cbramod_2024_eeg_fm.pdf)**: Criss-cross transformer for EEG decoding.
* **[Neuro-GPT (2024)](./pdfs/neuro_gpt_2024.pdf)**: GPT-based foundation for neural signals.
* **[BrainGFM (2024)](./pdfs/braingfm_2024.pdf)**: Graph-based BFM for heterogeneous fMRI.
* **[Brain-JEPA (2024)](./pdfs/brain_jepa_2024.pdf)**: Joint-Embedding Predictive Architecture for brain dynamics.
* **[BrainLM (2024)](./pdfs/brain_lm_2024.pdf)**: Language modeling on brain signals.

### 2.3 Mamba & State-Space Models (SSM) in Neuro

* **[Mentality (2025)](./pdfs/mentality_eeg_mamba_2025.pdf)**: Mamba-based BFM for EEG.
* **[EEGMamba (2024)](./pdfs/eegmamba_2024.pdf)**: Mixture-of-Experts Mamba for universal EEG classification.
* **[BrainMamba (2024)](./pdfs/brain_mamba_2024.pdf)**: Attention-free framework for brain activity encoding.
* **[Mamba Foundation (2024)](./pdfs/mamba_foundation_2024.pdf)**: The original Mamba paper for hardware/algo foundations.

### 2.4 Digital Twin Brain (DTB)

* **[SNN Algorithm (2025)](./pdfs/dfmg_2025_snn_algo.pdf)**: Delay-flush multi-group algo for scalable SNN simulation.

### 2.5 Analog & Neuromorphic Hardware
