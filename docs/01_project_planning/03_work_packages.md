# Work Packages (WP)

WP는 **평가항목에 직접 대응**하도록 구성하고, 각 WP는 “입력→방법→산출→검증”이 닫히도록 설계합니다.

## WP 초안(예시)

- **WP1: 데이터/표현 기반 구축**
  - 목표: Connectome/뇌 데이터의 표현(그래프/임베딩/제약조건)을 정의
  - 산출물: 데이터 사양서, 전처리 파이프라인, 품질 기준
- **WP2: Brain-inspired 모델/학습**
  - 목표: connectome-derived bias를 모델에 주입(구조/정규화/학습규칙)
  - 산출물: 모델 baseline + ablation + 재현 코드/리포트
- **WP3: 효율화 및 Neuromorphic 로드맵**
  - 목표: 저전력/저지연을 위한 압축/스파스/이벤트 기반 전환 전략
  - 산출물: 효율 지표, 단계별 전환 계획, co-design 실험
- **WP4: 검증/데모/확산**
  - 목표: 벤치마크/데모 시나리오/기술이전 경로
  - 산출물: 데모, 성능 보고서, 성과(논문/특허/표준) 계획
