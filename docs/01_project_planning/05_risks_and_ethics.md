## 흔한 리스크 체크

- **데이터/윤리**: IRB, 개인정보/민감정보, 데이터 반출/보관/접근통제, 익명화
- **재현성**: 실험 조건/버전 고정, 데이터 스플릿/프로토콜 문서화
- **실행가능성**: 연차별 범위 과대, 인력/장비/협력 기관 리스크
- **성과 패키징**: 논문/특허/데모/기술이전의 타임라인 불일치

## Neuromorphic 확장 리스크(예시)

- 알고리즘 효과는 있는데 HW 매핑이 어려움 → **단계적 지표(에너지/지연)로 분해** + co-design 최소 PoC 정의
