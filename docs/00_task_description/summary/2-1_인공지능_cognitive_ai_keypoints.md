## 1) 우리가 집중해야 할 품목(관리번호)

### 2026-인공지능-010 (총괄/세부1 포함, 병렬형)

- **총괄과제명**: 인간 뇌 인지 메카니즘을 모사한 감각-기억-학습-추론 통합 지능 원천 기술 개발
- **세부1(총괄/세부1)**: 다층적 상황 이해와 창의적 문제 해결을 위한 인간인지모사 기반 통합 추론 기술 개발
- **핵심 포인트**
  - 감각/학습/기억/추론 정보를 **통합**해 “다층적 상황 이해 + 창의적 문제 해결” 가능한 통합 AI
  - **적응적 추론 벤치마크 부재**를 직접 해결(새 벤치마크/성능지표 제시 및 검증 요구)
  - 메타인지(자기 검증 루프)로 **환각 최소화/설명가능성 강화** 방향 명시
- **기간/예산(가이드)**: 4년(2+2), `’26년 31억원 이내`, 총 `15,550백만원 이내`
- **TRL/연구유형**: TRL 2~4, 기초연구 표시(원천)

### 2026-인공지능-011 (세부2, 병렬형)

- **세부2**: 뇌 인지과학과 생물학적 학습 원리에 기반한 인공지능 학습 기술 개발
- **핵심 포인트**
  - 역전파 중심에서 벗어나 **국소 신호 + 보상 신호** 기반 학습(생물학적/인지과학적 원리 모사)
  - 목표 지향: **데이터 효율성 / 지속학습(망각 완화) / 적응성 / 에너지 효율성**을 동시에
  - “총체적 평가 지표 및 벤치마크” 개발 요구
- **기간/예산(가이드)**: 4년(2+2), `’26년 23억원 이내`, 총 `11,450백만원 이내`
- **TRL/연구유형**: TRL 2~4, 기초연구

### 2026-인공지능-012 (세부3, 병렬형)

- **세부3**: 인간 인지 원리를 반영한 인공지능 기억 메커니즘 기술 개발
- **핵심 포인트**
  - 사건(episode) 단위 분절 + 시공간 맥락 보존 + **의미기억(semantic memory) 일반화**
  - **망각/강화**로 기억 자원 효율적 관리, 멀티모달 통합 표현
  - 회상 정확도/맥락 유지력/의미 유지력/설명가능성 등 **기억 성능 벤치마크** 요구
- **기간/예산(가이드)**: 4년(2+2), `’26년 23억원 이내`, 총 `11,450백만원 이내`
- **TRL/연구유형**: TRL 2~4, 기초연구

### 2026-인공지능-013 (세부4, 병렬형)

- **세부4**: 뇌 신경계 원리 기반의 통합 감각 인공지능 모델 개발
- **핵심 포인트**
  - 뇌 감각 시스템의 **다중감각 통합 + 개념 주도(concept-driven, top-down) 처리 + 선택적 주의**
  - 시각-언어 중심 멀티모달을 넘어 촉각/고유감각/전정감각 등 **물리 감각 통합** 강조
  - 실세계 모호성/불완전성에 강건한 지각·인지 판단 + 벤치마크/평가 프레임 개발 요구
- **기간/예산(가이드)**: 4년(2+2), `’26년 23억원 이내`, 총 `11,450백만원 이내`
- **TRL/연구유형**: TRL 2~4, 기초연구

### 2026-인공지능-014 (세부5, 병렬형)

- **세부5**: 인공지능을 위한 인간 인지 기반의 데이터 의미화 및 표현 기술 개발
- **핵심 포인트**
  - 무차별 수집/정제에서 벗어나 **주의·목표 기반 선택적 수집**, **의미 연결망(semantic linkage)** 조직
  - 반복 패턴으로부터 **개념 단위 추상화/일반화**, 가치·맥락 기반 동적 저장/갱신/망각
  - “인지형 데이터 의미 표현 구조 구현 및 벤치마크” 요구
- **기간/예산(가이드)**: 4년(2+2), `’26년 23억원 이내`, 총 `11,450백만원 이내`
- **TRL/연구유형**: TRL 2~4, 기초연구

## 2) 공통 “필수 운영조건”(제안서에서 반드시 대응)

- **총괄-세부 연계/통합**: 총괄(010)이 011~014의 수행체계를 연계/통합하고 **통합 시나리오**를 제시해야 함
- **통합 벤치마크/성능지표**: 통합 인공지능을 평가할 **새 벤치마크 + 성능지표**를 제시/검증 요구
- **데이터셋 연계/통합**: 총괄/세부 과제 간 데이터셋을 연계·통합 수행 요구
- **연차별 성과 공유/확산 절차**: 매년 전체 성과 공유와 확산(오픈) 프로세스 제안 및 수행
- **다학제 공동연구 필수**: 뇌과학/인지과학/심리학/언어학 등 공동연구 참여 “필수”
- **오픈 정책(표기)**: 공개SW, 기술료비징수, 연구데이터 공개 등 조건이 다수 품목에 명시됨

## 3) SNU Connectome 관점의 “핵심 제안 메시지”(초안)

- **Brain-to-AI inductive bias**: connectome(구조/기능) 기반 제약을 통해 “스케일”이 아닌 “구조/학습규칙/기억/주의”로 성능과 효율을 동시에 노림
- **통합형 지능**: 감각→의미화→기억→학습→추론을 단일 아키텍처/프로토콜로 연결(벤치마크 포함)
- **Neuromorphic 로드맵**: 국소 학습·스파스·이벤트 기반(주의/감각) 요소를 통해 장기적으로 neuromorphic 매핑 가능성 제시
