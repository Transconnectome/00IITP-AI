## 1) 폴더 구조 및 내용

- **`./` (현재 위치)**: 핵심 RFP 및 공고문
- **`raw/`**: 정부 배포 원본 파일 전체
  - `(붙임 4) 신청서식 및 참고자료/`: **연구개발계획서 양식** 등 필수 제출 서류
  - `(붙임 5) 국가연구개발혁신법.../`: 관련 법령 및 규정
  - `(붙임 6) IRIS 전산접수 매뉴얼/`: 접수 시스템 가이드
- **`summary/`**: 과제 내용 요약 노트 (AI 분석 결과 등)
- **`parsed/`**: PDF/HWP 텍스트 추출본 (LLM 분석용)

## 3) 체크리스트 (요구사항 → 실행 항목)

- **제안서 분량/서식**: 글꼴/줄간격/페이지 제한/그림 해상도/서명 여부
- **필수 섹션**: 목표/필요성/차별성/연구내용/WP/추진체계/예산/파급효과/리스크
- **정량 지표**: KPI/마일스톤/TRL/성능·효율 목표
- **윤리·보안·데이터**: IRB, 개인정보/민감정보, 데이터 반출/보관, 보안 등급
- **기술 범위 제약**: 제외/권장 기술, 필수 데이터/플랫폼, 표준/상호운용
- **평가 매핑**: “우리 주장/근거/실행”이 평가항목에 어떻게 대응되는지
