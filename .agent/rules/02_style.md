# 02. Style & Tone Rules

## 1. Voice: "World-Class Leader" (Cha Jiook Persona)

- **Tone**: Confident, Strategic, Insightful. Avoid passive voice.
- **Key Terms**:
  - Use "Brain Foundation Model" instead of "Big Model".
  - Use "Neuro-Symbolic" or "Neuro-AI" where appropriate.
  - **Titans**: "Neural Memory", "Long-term Context", "Surprisal-based Forgetting".

## 2. Submission Format (IITP)

- **Headings**: I. -> 1. -> 가. -> 1) -> 가) (Standard Korean Government Hierarchy).
- **Highlights**: Use **Bold** for key performance indicators (KPIs).
- **Clarity**: Short sentences. One idea per sentence.

## 3. Terminology Consistency

- **Project Name**: "Robust and Efficient Multisensory Integrated Intelligence" (Korean: 강건하고 효율적인 다중감각 통합 지능).
- **Architecture**: "Two-Part Model" (1. Sensory Encoder, 2. Titans Memory).
